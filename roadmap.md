# Roadmap

## Phase 1 — MVP ✅

- [x] Project setup (HTML/CSS/JS scaffold)
- [x] **Welcome page** (logo/branding, tagline, "Enter" button to transition into the desktop)
- [x] macOS desktop background (Sequoia wallpaper)
- [x] Top menu bar (Apple logo, time, battery icon)
- [x] Dock at the bottom (real macOS icons, Photo Booth highlighted)
- [x] Dock position setting — user can move dock to left, right, or bottom
- [x] Background wallpaper picker — 6 preset macOS wallpapers + custom upload
- [x] Photo Booth window frame (title bar, traffic lights, correct styling)
- [x] Webcam integration via `getUserMedia`
- [x] Mirror effect on the video feed (flipped horizontally, like real Photo Booth)
- [x] Capture button (red shutter button, centered below the feed)
- [x] Countdown animation (3-2-1) before capture
- [x] Screen flash effect on capture
- [x] Filmstrip overlay at bottom of camera area (glass aesthetic)
- [x] Photo preview via lightbox + download

## Phase 2 — Polish ✅

- [x] Photo Booth effects panel (9 effects: Mirror, Thermal, Sepia, B&W, Pop Art, Invert, Cool)
- [x] 3x3 live effects grid with cover-crop rendering
- [x] Smooth transitions entering/exiting effects view
- [x] Collage mode (4-photo 2x2 grid with consecutive captures)
- [x] Video mode easter egg
- [x] Realistic window drag behavior (generic, reusable)
- [x] Window z-index stacking (click to focus)
- [x] Dock magnification hover effect
- [x] Photo deletion from filmstrip (hover X)
- [x] System Settings as draggable window (opened from dock)
- [x] Help window (accessible from menu bar + settings)
- [ ] Boot-up animation on first load (Apple logo → desktop)
- [ ] Cursor set to macOS default pointer

## Phase 3 — Share & Grow

- [ ] Share to social (Twitter/X, Instagram story format)
- [ ] Watermark option ("taken on idonthaveamac.com")
- [ ] OG meta tags for link previews
- [ ] Analytics (simple, privacy-respecting)
- [ ] Custom domain setup (idonthaveamac.com)
