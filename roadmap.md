# Roadmap

## Phase 1 — MVP

- [ ] Project setup (HTML/CSS/JS scaffold)
- [ ] **Welcome page** (logo/branding, tagline, "Enter" button to transition into the desktop)
- [ ] macOS desktop background (Sequoia wallpaper)
- [ ] Top menu bar (Apple logo, time, battery icon)
- [ ] Dock at the bottom (static icons, Photo Booth highlighted)
- [ ] Dock position setting — user can move dock to left, right, or bottom
- [ ] Background wallpaper picker — preset macOS wallpapers + custom upload
- [ ] Photo Booth window frame (title bar, traffic lights, correct styling)
- [ ] Webcam integration via `getUserMedia`
- [ ] Mirror effect on the video feed (flipped horizontally, like real Photo Booth)
- [ ] Capture button (red shutter button, centered below the feed)
- [ ] Countdown animation (3-2-1) before capture
- [ ] Screen flash effect on capture
- [ ] Display captured photo in a strip below the feed (like Photo Booth's filmstrip)
- [ ] Download button for the captured photo

## Phase 2 — Polish

- [ ] Photo Booth effects panel (sepia, black & white, thermal, x-ray, etc.)
- [ ] Multi-shot burst mode (4-photo grid like classic Photo Booth)
- [ ] Realistic window drag behavior
- [ ] Dock magnification hover effect
- [ ] Boot-up animation on first load (Apple logo → desktop)
- [ ] Cursor set to macOS default pointer

## Phase 3 — Share & Grow

- [ ] Share to social (Twitter/X, Instagram story format)
- [ ] Watermark option ("taken on idonthaveamac.com")
- [ ] OG meta tags for link previews
- [ ] Analytics (simple, privacy-respecting)
- [ ] Custom domain setup (idonthaveamac.com)

## Phase 4 — Bonus Ideas

- [ ] FaceTime window variant
- [ ] iMessage conversation window (fake texts)
- [ ] "About This Mac" modal showing the user's actual browser/OS info
- [ ] Dark mode toggle via menu bar
